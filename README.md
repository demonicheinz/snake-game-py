##  Cara Menjalankan:

1. Pastikan Pygame terinstall:
    ```bash
    pip install pygame
    ```
2. Jalankan di terminal:
    ```bash
    python snake_game.py
    ```

## Kontrol:

- ⬅️ Kiri → Left Arrow
- ➡️ Kanan → Right Arrow
- ⬆️ Atas → Up Arrow
- ⬇️ Bawah → Down Arrow
